Цикл з параметром

For1. Дані цілі числа Kі N (N > 0). Вивести N раз число K.

For2. Дано два цілих числа A та B (A < B). Вивести в порядку зростання всі цілі числа, розташовані між A і B (включаючи самі числа AA і B< /i>), а також кількість N цих чисел.

For3. Дано два цілих числа A та B (A < B). Вивести в порядку зменшення всі цілі числа, розташовані між A і B (не включаючи числа A і B) ), а також кількість N цих чисел.

For4. Дано дійсне число— ціна 1 кг цукерок. Вивести вартість 1, 2, 10 кг цукерок.

For5*. Дано дійсне число— ціна 1 кг цукерок. Вивести вартість 0.1, 0.2, 1 цукерок.

For6. Дано дійсне число— ціна 1 кг цукерок. Вивести вартість 1.2, 1.4, 2 цукерок.

For7. Дано два цілих числа A та B (A < B). Знайти суму всіх цілих чисел від  A до B включно.

For8. Дано два цілих числа A та B (A < B). Знайти добуток усіх цілих чисел від A до B включно.

For9. Дано два цілих числа A та B (A < B). Знайти суму квадратів всіх цілих чисел від A до B включно.

For10. Дано ціле число NN (> 0). Знайти суму

1 + 1/2 + 1/3 + … + 1/N

(речове число).

For11. Дано целое число N (> 0). Найти суму

N2 + (N + 1)2  + (N + 2)2 + … + (2&# 183;N)2

(ціле число).

For12*. Дано целое число N (> 0). Знайти твори

1.1 · 1.2 · 1.3 · …

(N сомножителей).

For13*. Дано целое число N (> 0). Знайти значення виражень

1.1 – 1.2 + 1.3 – …

(N слагаемых, знаки чередуются). Условный оператор не использовать.

For14. Дано целое число N (> 0). Знайдіть квадрат цього числа, використовуючи для його вирахування наступну формулу:

N2 = 1 + 3 +  ;5 + … + (2·N – 1).

Після додавання до суми кожного вкладеного виводиться поточне значення суми (в результаті будуть виведені квадрати всіх цілих чисел від 1 до N).

For15*. Дано речовинне число A і ціле число N (> 0). Найти A в ступені N:

AN< /sup> = A·A· … ·A

(числа A перемножуються N раз).

For16*. Дано речовинне число A і ціле число N (> 0). Використовуючи один цикл, виведіть усі цілі ступені числа A від 1 до N.

For17. Дано речовинне число A і ціле число N (> 0). Використовується один цикл, знайдіть суму

1 + A + A2 +  A3 + … + AN.

For18. Дано речовинне число A і ціле число N (> 0). Використовується один цикл, знайдіть значення виражень

1 – A + A2 – A3 + … + (–1)< i>N·AN.

Условный оператор не использовать.

For19*. Дано целое число N (> 0). Найти твори

N! = 1·2·…·N

(N–факторіал). Щоб уникнути цілочисленного переповнення, вичислити це творіння з допомогою матеріальної змінної та вивести його як матеріальне число.

For20*. Дано целое число N (> 0). Використовується один цикл, знайдіть суму

1! + 2! + 3! + … + N!

(виражение N! — N–факторіал  ;— обозначает произведение всех целых чисел от 1·2· …·N). Щоб уникнути цільового переповнення, здійсніть збільшення з допомогою матеріальних змінних і виведіть результат як матеріальне число.

For21. Дано ціле число NN (> 0). Використовуючи один цикл, знайти суму

1 + 1/(1!) + 1/(2!) + 1/(3!) +  ;… + 1/(N!)

(вираз N! &# 151; N–факторіал — ;N! = 1·2·…·N). Отримане число є наближеним значенням константи e = exp(1).

For22. Дано дійсне число X та ціле число N (> 0). Знайти значення виразу

1 + X + X2/(2!)  ;+ … + XN/(N!)

(N! = 1·2·…·N) . Отримане число є наближеним значенням функції exp в точці X.

For23. Дано дійсне число X та ціле число N (> 0). Знайти значення виразу

X – X3/(3!) +  X5/(5!) – … + (–1) N·X2·N+1/((2&# 183;N+1)!)

(N! = 1·2· …·N). Отримане число є наближеним значенням функції sin в точці X.

For24. Дано дійсне число X та ціле число N (> 0). Знайти значення виразу

1 – X2/(2!) + X4/(4!) – … + (–1)N ·X2·N/((2·N )!)

(N! = 1·2·…·N ). Отримане число є наближеним значенням функції cos в точці X.

For25. Дано дійсне число X (|X| < 1) і ціле число N (> 0 ). Знайти значення виразу

X – X2/2 + X3/3 – … + (–1)N –1·XN/N.

Отримане число є наближеним значенням функції ln в точці 1 + X.

For26. Дано дійсне число X (|X| < 1) і ціле число N (> 0 ). Знайти значення виразу

X – X3/3 + X5/5 – … + (–1)N ·X2·N+1/(2·N+1).

Отримане число є наближеним значенням функції arctg в точці X.
