Цикл з умовою

While1*. Дано позитивні числа A і B (A > B). На відрізку довжини A розміщено максимально можливу кількість відрізків довжини B (без накладень). Не використовуючи операції множення та поділу, знайти довжину незайнятої частини відрізка A.

While2*. Дано позитивні числа A і B (A > B). На відрізку довжини A розміщено максимально можливу кількість відрізків довжини B (без накладень). Не використовуючи операції множення та поділу, знайти кількість відрізків B, розміщених на відрізку A.. .

While3. Дано цілі позитивні числа N і K. Використовуючи тільки операції додавання та віднімання, знайти приватне відокремлення націло на N на K, а також залишок від цього поділу.

While4*. Дано ціле число NN (> 0). Якщо воно є ступенем числа 3, то вивести True, якщо не є — вивести False.

While5. Дано ціле число N (> 0), що є деяким ступенем числа 2: N = = 2K. Знайти ціле число K — показник цього ступеня.

While6. Дано ціле число NN (> 0). Знайти подвійний факторіал N:

N!! = N ·(N–2)·(N–4)·…

(останній співмножник дорівнює 2, якщо N — парне, і 1, якщо N — ; непарне). Щоб уникнути цілісного переповнення, обчислювати цей твір за допомогою речової змінної і вивести його як речове число.

While7*. Дано ціле число NN (> 0). Знайти найменше ціле позитивне число K, квадрат якого перевищує N: K2 >  N. Функцію вилучення квадратного кореня не використовувати.

While8. Дано ціле число NN (> 0). Знайти найбільше ціле число K, квадрат якого не перевищує N: K2 Ј N. Функцію вилучення квадратного кореня не використовувати.

While9. Дане ціле число NN (> 1). Знайти найменше ціле число K, при якому виконується нерівність 3K > N.< /p>

While10. Дане ціле число NN (> 1). Знайти найбільше ціле число K, при якому виконується нерівність 3K < N.< /p>

While11*. Дане ціле число NN (> 1). Вивести найменше з цілих чисел K, для яких сума 1 + + … + K буде більшою або дорівнює  ;N, і саму цю суму.

While12*. Дане ціле число NN (> 1). Вивести найбільше з цілих чиселK, для яких сума 1 + + … + K буде меншою або дорівнює  ;N, і саму цю суму.

While13. Дано число A (> 1). Вивести найменше з цілих чиселK, для яких сума 1 + 1/2 + … + 1/K буде більше A, і саму цю суму.

While14. Дано число A (> 1). Вивести найбільше з цілих чиселK, для яких сума 1 + 1/2 + … + 1/K буде менше A, і саму цю суму.

While15. Початковий внесок у банку дорівнює 1000 руб. Через кожен місяць розмір вкладу збільшується на P відсотків від наявної суми (P — дійсне число, 0 <  P < 25). За даним  P визначити, через скільки місяців розмір вкладу перевищить 1100 руб., і вивести знайдену кількість місяців K (ціле число) і підсумковий розмір вкладу  S (речове число).

While16. Спортсмен-лижник почав тренування, пробігши в перший день 10 км. Кожен наступний день він збільшував довжину пробігу наP відсотків від пробігу попереднього дня (P — речовинне, 0 < < i>P < 50). По даному визначити, після якого дня сумарний пробіг лижника за всі дні перевищить 200 км, і вивести знайдену кількість днів (K) (ціле) і сумарний. пробіг S (речове число).

While17. Дано ціле число NN (> 0). Використовуючи операції поділу націло і взяття залишку від поділу, вивести всі його цифри, починаючи з правої (розряду одиниць).

While18. Дано ціле число NN (> 0). Використовуючи операції поділу націло і взяття залишку від поділу, знайти кількість і суму його цифр.

While19. Дано ціле число NN (> 0). Використовуючи операції поділу націло і взяття залишку від ділення, знайти число, отримане при прочитанні числа N справа наліво.

While20. Дано ціле число NN (> 0). За допомогою операцій поділу націло і взяття залишку від поділу визначити, чи є в запису числа N цифра «2». Якщо є, то вивести True, якщо ні  — вивести False.

While21. Дано ціле число NN (> 0). За допомогою операцій поділу націло і взяття залишку від поділу визначити, чи є в записі числа непарні цифри. Якщо є, то вивести True, якщо немає — вивести False.

While22*. Дане ціле число NN (> 1). Якщо воно є простим, то є не позитивних дільників, крім 1 і самого себе, то вивести True, інакше вивести False.

While23*. Дано цілі додатні числа A та B. Знайти їх найбільший спільний дільник, використовуючи алгоритм Евкліда.

While24. Дане ціле число NN (> 1). Послідовність чисел Фібоначчі FK визначається таким чином:

F 1 = 1,        F2 = 1,        FK = FK& #150;2 + FK–1,    K = 3, 4, … .

Перевірити, чи є чи число < i>N числом Фібоначчі. Якщо є, то вивести True, якщо ні  — вивести False.

While25. Дане ціле число NN (> 1). Знайти перше число Фібоначчі, більше N. (визначення чисел Фібоначчі дано у завданні While24).

While26. Дано ціле число N (> 1), що є числом Фібоначчі: N = FK (визначення чисел Фібоначчі дано в завданні While24). Знайти цілі числа FK–1 і FK< /i>+1 — попереднє і наступне число Фібоначчі.

While27. Дано ціле число N (> 1), що є числом Фібоначчі: N = FK (визначення чисел Фібоначчі дано в завданні While24). Знайти ціле число K — порядковий номер числа Фібоначчі N.

While28. Дано дійсне число e (> 0). Послідовність дійсних чисел AK визначається так:

A1< /sub> = 2,        AK = 2 + 1/AK–1,    K = 2, 3, … .

Знайти перший з номерів K, для яких виконується умова |AK – AK–1| < e, і вивести цей номер, а  також числа AK–1 і AK.

While29. Дано дійсне число e (> 0). Послідовність дійсних чисел AK визначається так:

A1< /sub> = 1,        A2 = 2,        AK = (AK –2 + 2·AK–1)/3,    K = 3, 4, … .

While30. Дано позитивні числа A, B, C. На прямокутнику розміру A ґ B розміщено максимально можливу кількість квадратів зі стороною  C (без накладень). Знайти кількість квадратів, розміщених на прямокутнику. Операції множення та поділу не використовувати.
